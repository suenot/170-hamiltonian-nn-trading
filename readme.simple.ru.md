# Гамильтоновы нейронные сети для трейдинга: простое объяснение

## Что такое гамильтониан? Представьте маятник!

### Маятник на старых часах

Представьте маятник, качающийся туда-сюда на старых настенных часах:

```
      O  (точка подвеса)
     /|
    / |
   /  |           Наверху: МЕДЛЕННО, много "запасённой" энергии
  *   |           (как мячик на вершине горки)
      |
      |    *      В середине: БЫСТРО, много "энергии движения"
      |           (как мячик, катящийся по дну горки)
      |
      |  *        На другой стороне: снова МЕДЛЕННО
      |           Вся "энергия движения" снова стала "запасённой"
```

И вот в чём магия: **ПОЛНАЯ энергия никогда не меняется!**

```
В любой момент:

  Энергия движения + Запасённая энергия = КОНСТАНТА

  (Учёные называют их "кинетическая" и "потенциальная" энергия)
```

Когда маятник наверху:
- Энергия движения = МАЛО (он еле двигается)
- Запасённая энергия = МНОГО (гравитация хочет его потянуть вниз)

Когда он проносится через низ:
- Энергия движения = МНОГО (он летит быстро!)
- Запасённая энергия = МАЛО (он в самой нижней точке)

Но сложите их -- и вы всегда получите одно и то же число. Всегда. Это называется **закон сохранения энергии**, и его открыл математик Уильям Гамильтон в 1830-х годах.

## Чем рынок похож на маятник?

### Качели цены и импульса

Подумайте о цене биткоина:

```
Фаза 1: Цена падает ниже средней
         "Запасённая энергия" накапливается (как если потянуть маятник в сторону)
         Охотники за скидками начинают покупать...

Фаза 2: Импульс покупок нарастает
         "Энергия движения" растёт (маятник проносится через середину)
         FOMO включается, все покупают...

Фаза 3: Цена перескакивает выше средней
         "Запасённая энергия" снова накапливается (маятник на другой стороне)
         Люди начинают фиксировать прибыль...

Фаза 4: Импульс продаж нарастает
         Цена начинает падать обратно...
         И цикл повторяется!
```

Нарисуем это:

```
Отклонение цены (насколько далеко от "нормы")
    ^
    |        * <-- Цена слишком высокая (запасённая энергия)
    |       / \
    |      /   \     * <-- Снова перескакивает
    |     /     \   / \
----+----/-------\-/---\----> Время
    |  /          *     \
    | /                  \
    |*                    * <-- Цена слишком низкая (запасённая энергия)
    |
```

Это выглядит ТОЧНО как качание маятника!

```
Маятник:                     Рынок:
Угол (положение) -------->   Отклонение цены (насколько далеко от средней)
Скорость (импульс) ------>   Скорость изменения цены (моментум)
Гравитация ---------->       Сила возврата к среднему
Сопротивление воздуха -->    Транзакционные издержки (замедляют всё)
```

## Что такое фазовый портрет?

Если отслеживать одновременно положение И скорость маятника в каждый момент и нарисовать их на графике, получится нечто красивое:

```
    Скорость (импульс)
        ^
        |     .---.
        |    /     \
        |   |   .   |     <-- Каждая петля -- это один полный взмах
        |    \     /
        |     '---'
        +----------------> Положение (отклонение)
```

Для рынка:

```
    Моментум цены (как быстро меняется цена)
        ^
        |      .---.
        |     /     \
        |    |   .   |    <-- Один рыночный цикл
        |     \     /
        |      '---'
        +-------------------> Отклонение цены (насколько далеко от нормы)
```

Если бы рынок был идеальным маятником, эти петли были бы идеальными кругами или эллипсами. В реальности они закручиваются внутрь (из-за трения/издержек) или иногда перескакивают на новые орбиты (смена режимов).

## Что такое гамильтонова нейронная сеть?

### Обычные нейросети vs HNN

**Подход обычной нейросети:**
"Эй, нейросеть, вот тебе данные о ценах. Просто предскажи, что будет дальше. Разберись сама."

Проблема: сеть НИЧЕГО не знает о сохранении энергии. Её предсказания со временем могут уплыть в бессмыслицу.

```
Предсказание обычной НС через много шагов:

  Цена может уйти:       Что происходит на самом деле:
      /                        /\    /\
     /                        /  \  /  \
    /   (дрейф в              /    \/    \
   /     бесконечность!)
```

**Подход гамильтоновой нейросети:**
"Эй, нейросеть, выучи ФУНКЦИЮ ЭНЕРГИИ этой системы. Я знаю, что полная энергия (примерно) сохраняется."

```
Предсказание HNN через много шагов:

  Что предсказывает HNN:   Что происходит на самом деле:
      /\    /\                   /\    /\
     /  \  /  \                 /  \  /  \
    /    \/    \                /    \/    \
   (остаётся в границах!)
```

### Как это работает?

```
Шаг 1: Подаём данные о цене (положение) и моментуме в нейросеть
        ┌──────────────────────────┐
  q --> │                          │
        │    Нейронная сеть        │ --> H (одно число: "энергия")
  p --> │                          │
        └──────────────────────────┘

Шаг 2: Используем математику (автодифференцирование), чтобы получить
        ЗАКОНЫ ДВИЖЕНИЯ из функции энергии

        dH/dp  = как меняется положение  (скорость)
       -dH/dq  = как меняется импульс    (сила)

Шаг 3: Используем эти законы для предсказания будущего!
        Начиная с сегодняшних (положение, импульс), шагаем вперёд по времени,
        следуя законам. Предсказания естественно сохраняют энергию.
```

Как будто сеть учит ЗАКОНЫ ФИЗИКИ рынка, а не просто запоминает шаблоны.

## Почему это лучше обычного предсказания?

### Аналогия с шаром для боулинга

Представьте, что вы хотите предсказать, куда покатится шар для боулинга:

**Способ А (Обычная НС):** Посмотреть на 1000 шаров. Запомнить каждую траекторию. Надеяться, что следующий шар сделает что-то похожее.

**Способ Б (HNN):** Выучить, что шары подчиняются гравитации и трению. Из этих ЗАКОНОВ предсказать, куда покатится ЛЮБОЙ шар, даже на дорожке, которую вы никогда не видели.

Способ Б мощнее, потому что он понимает ПРАВИЛА, а не просто примеры.

Для рынков:
- **Способ А** ломается, когда рынок входит в новый режим
- **Способ Б** всё ещё работает, потому что базовая энергетическая динамика похожа

## Диссипативное расширение: у рынков есть трение!

Настоящий маятник в конце концов останавливается из-за сопротивления воздуха. У рынков своё собственное "сопротивление воздуха":

```
Идеальный маятник (без трения):     Реальный маятник (с трением):

    Скорость                            Скорость
      ^                                   ^
      |  .---.                            |  .---.
      | |     |                           | |  .--.
      | |  .  |  (крутится вечно)         | | | .. |  (закручивается внутрь)
      | |     |                           | |  '--'
      |  '---'                            |  '---'
      +---------> Положение               +---------> Положение
```

Для рынков "трение" -- это:
- Транзакционные издержки (каждая сделка стоит денег)
- Спред bid-ask (вы всегда покупаете дорого и продаёте дёшево)
- Затухание информации (старые паттерны тускнеют)
- Рыночное воздействие (крупные сделки двигают цену против вас)

**Диссипативная HNN** добавляет член трения:

```
Изменение энергии = -трение * скорость

Вместо: полная энергия = константа
Получаем: полная энергия медленно убывает (как на реальном рынке)
```

Это делает модель гораздо реалистичнее!

## Реальный пример: торговля биткоином на Bybit

### Шаг 1: Получаем данные

Скачиваем данные о цене и объёме биткоина с биржи Bybit.

### Шаг 2: Создаём "фазовое пространство"

```
Из сырых данных:               В фазовое пространство:

  Цена: $50,000                  q = насколько цена отклонилась от 20-периодной средней
  Объём: 1000 BTC                p = как быстро меняется цена (моментум)
  Время: 10:00
```

### Шаг 3: Обучаем HNN

Нейронная сеть выучивает функцию энергии H(q, p). После обучения она знает:
- Сколько "энергии" у рынка сейчас
- Как моментум и отклонение цены обмениваются друг с другом
- Какой естественный период колебаний

### Шаг 4: Делаем предсказания

```
Текущее состояние: q = +0.02 (цена на 2% выше средней), p = -0.01 (падает)

HNN предсказывает: "Энергия преобразуется из 'запасённой' (отклонение цены)
                    в 'кинетическую' (моментум). Цена продолжит падение
                    к средней, слегка проскочит, потом отскочит обратно."

Торговый сигнал: ПРОДАЖА (цена упадёт к средней)
```

### Шаг 5: Управление рисками

```
Если "энергия" вдруг подскакивает (как будто маятник пнули):

  Энергия
    |          * <-- Резкий скачок! Что-то изменилось!
    |     ****
    | ****
    |*
    +-----------> Время

Сигнал: "СМЕНА РЕЖИМА -- сокращайте позиции!"
```

Это одна из самых крутых особенностей HNN -- она может обнаружить, когда "физика" рынка меняется.

## Что делает это особенным?

```
                    Обычная НС       HNN
                    ─────────         ───
Долгие прогнозы:    Уплывают          Остаются в границах
Физика:             Нет               Сохранение энергии
Понятность:         Чёрный ящик       "У рынка X энергии"
Смена режимов:      Трудно поймать    Естественно (скачки энергии)
Эффективность:      Нужно много данных Работает с меньшим количеством
                                      (физика заполняет пробелы)
```

## Итоги

1. **Рынки колеблются** как маятники -- цена и моментум обмениваются друг с другом
2. **Гамильтоновы нейросети** выучивают "функцию энергии" рынка
3. **Сохранение энергии** удерживает предсказания стабильными на длинных горизонтах
4. **Диссипация** учитывает реальное трение (издержки, спреды)
5. **Фазовые портреты** дают красивое визуальное понимание рыночной динамики
6. **Энергетические аномалии** естественно обнаруживают смену режимов

Думайте об этом так: вместо того чтобы пытаться предсказать, где будет каждая волна в океане, HNN выучивает ПРАВИЛА, по которым работают волны. Это гораздо более мощный способ понять рынок.
